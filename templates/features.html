Features

The bot itself is packed with features to keep people engaged in using the bot for longer periods of time.
Here is some of the things Comet offers:

Super Snipe
Comet supports the ability to save three deleted messages at any given moment. This ability is called Super Snipe.
To access this feature you do #SuperSnipe (1-3) on a server where comet is available.

To use regular snipe which only stores the most recently deleted message, you do #snipe
TTS with Unlimited Character Limit
Comet supports text-to-speech in any voice channel thanks to gTTS.
Due to this, Comet also has the ability to speak the text inputted in five different languages.
These languages are English, Spanish, Tagalog (Filipino), Armenian, and Korean.
To use TTS, you run #tts (text to speak) on a server where comet is available. The bot will then prompt the user to choose their language and the bot will shortly speak the text inputted in that language.
