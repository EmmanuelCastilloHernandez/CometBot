Features

The bot itself is packed with features to keep people engaged in using the bot for longer periods of time.
Here is some of the things Comet offers:

Super Snipe and Image Snipe
Comet supports the ability to save three deleted messages at any given moment. This ability is called Super Snipe. To access this feature you do #SuperSnipe (1-3) on a server where comet is available.

Image snipe is also a feature of Comet, where the bot is able to snipe images with the regular #snipe command that only stores the most recently-deleted message. To make Image Snipe be more private, the images get deleted once the bot restart of after 120 seconds have passed. This 120-second timer also applies for Super Snipe.

Edit Snipe
Comet supports the ability to display a message's content prior to when it was edited. This is called Edit Snipe. To use Edit Snipe, you user the command #edit to display a message before and after it was edited. Note that Edit Snipe clears after 80 seconds have gone by.

TTS with 1200-Character Limit
Comet supports text-to-speech in any voice channel thanks to gTTS. Due to this, Comet also has the ability to speak the text inputted in different languages. These languages are English, Spanish, Tagalog (Filipino), Armenian, Korean, Russian, and many more shown in the screenshot below.
To use TTS, you run #tts (text to speak) on a server where comet is available. The bot will then prompt the user to choose their language and the bot will shortly speak the text inputted in that language. Note that long pieces of text will take longer to process.

Zalgo Generation

To do this run #zalgo (text to curse) in a Comet-enabled server and Comet will generate the text in a message.
Music Player

The bot itself tries to play music to the best of its ability. Use #play to play music from YouTube, #ql to check the queue, #remove to remove a song from the queue, #q to queue a song, #pause to pause the player, #skip to skip the current song, and #leave to make Comet leave the Voice Channel. As of now, support for Spotify links is in progress
